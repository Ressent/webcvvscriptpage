## 🛠️ Personalización

### Eventos
Editar el array `eventosData` en `script.js` para actualizar las reuniones.

### Redes sociales
Modificar los atributos `href` en los enlaces de `.redes-grid` en `index.html`.

### Información de contacto
Actualizar datos en la sección Contacto y en el footer.

## 🚀 Despliegue en Netlify

1. Subir estos archivos a un repositorio en GitHub
2. Conectar Netlify al repositorio
3. Netlify detectará automáticamente la configuración

## 📁 Estructura

- `index.html` - Estructura principal
- `style.css` - Todos los estilos
- `script.js` - Funcionalidad y datos dinámicos
- `netlify.toml` - Configuración para Netlify
